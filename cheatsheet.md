# Jinx Cheatsheet

## Actions

>success = ability + 1d20

## Combat

### Unchallenged

>success = ability + modifiers + 1d20 **- 25**

### Challenged

>success = ability + modifiers + 1d20 **- adversary**

### Modifiers

size of target|modifier
---|---:
Tiny|-10
Small|-5
Medium|0
Big|+5
Huge|+10

### Damage

>damage = weapon * damage modifier

### Damage Modifier

success|damage modifier

31-40|2
41-50|2.5
